# PLCExamples

1. 클론 받으시고 유니티 허브로 프로젝트 추가하고 열기.
2. 에디터 상단 메뉴 Window -> Package Manager 클릭해 패키지 매니저 윈도루 열기.
3. 왼쪽 탭에서 My Assets 버튼 클릭.
4. Dotween을 검색하거나 리스트에서 DOTween (HOTween v2) 다운로드 및 임포트하기.
5. Free Outline 검색하거나 리스트에서 Free Outline 다운로드 및 임포트하기.

※ Dotween 임포트시 Utility Panel이 열리면 Setup DOTween.. 버튼 클릭하고 기다리면
하단에 Apply버튼 활성화되고 Apply 버튼 클릭.
그리고 Add ASMDEF 버튼 클릭.
